Daydreams is a framework for building generative agents that can execute tasks
over long time horizons.

## What is a generative agent?

Daydreams is designed to be a simple and as powerful as possible, with most of
the piping and orchestration handled for you by the LLMs reasoning.

## Features

- 🔗 Chain Agnostic: Execute transactions and interact with any blockchain
  network
- 👥 Multi-Expert System: Leverage specialized modules working together to solve
  complex tasks
- 🧠 Context Management: Simple yet powerful memory and context handling system
- 🎯 Goal-Oriented: Long-term planning and goal-oriented behavior capabilities
- 💾 Persistent Memory: Built-in support for storing and retrieving long-term
  information
- 🤔 Advanced Reasoning: Multi-step reasoning using Hierarchical Task Networks
